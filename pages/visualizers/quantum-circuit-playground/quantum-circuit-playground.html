⚛️ Understanding Grover's Speedup

🔍

Classical Search: O(N)

Searching an unsorted list of N items classically requires checking up to N items in the worst case — no way around it without extra structure.

⚡

Grover's Search: O(√N)

Grover's algorithm finds the marked item in roughly √N oracle calls by amplifying the marked state's amplitude each iteration via the diffusion operator — a quadratic, not exponential, speedup, but the best possible for unstructured search.

🎯

The Oracle + Diffusion Cycle

Each Grover iteration: the oracle flips the phase of the target state, then the diffusion operator reflects all amplitudes about their average — geometrically rotating the state vector toward the target.

🔗

Entanglement

A CNOT gate applied to a qubit in superposition creates entanglement — a state that cannot be described as independent qubits. Measuring one instantly determines the other, regardless of distance.
