    How Constraint Mapping Works

    01

    Estimate the budget

    Rough budget ≈ time limit × ops/sec. With 2s and 10⁸ ops/s you get about 2×10⁸ safe operations.

    02

    Plug in n (and m)

    Compare O(n), O(n log n), O(n√n), O(n²), … against the budget. Prefer the simplest complexity that still fits.

    03

    Watch memory too

    Large arrays / matrices can blow memory before time. A few ints per element is fine; O(n²) tables often are not.
